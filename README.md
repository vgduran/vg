# Landing Page de Víctor Duran

Bienvenido al repositorio de la landing page de Víctor Duran.

## ¿Qué es esto?
Este repositorio contiene los archivos para la página web pública de Víctor Duran, Ingeniero, Instructor y Emprendedor.

## Tecnologías
- HTML puro
- CSS incrustado para diseño
- Alojado en GitHub Pages

## Contenido:
- `index.html`: Es la página principal (landing page).
- `README.md`: Este archivo, con información sobre el proyecto.
- `.github/workflows/deploy-pages.yml`: Configuración para que GitHub Pages se actualice automáticamente.

## Cómo actualizar la página:
1.  Haz clic en el archivo `index.html` en GitHub.
2.  Haz clic en el icono del lápiz para editar.
3.  Realiza tus cambios en el código HTML.
4.  Desplázate hacia abajo, escribe un mensaje de commit (ej: "Update hero section") y haz clic en "Commit changes".

GitHub Pages se actualizará automáticamente en unos minutos.
